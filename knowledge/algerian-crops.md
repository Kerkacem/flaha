# قاعدة بيانات المحاصيل الجزائرية

## القمح (Blé)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | شتوي (أكتوبر-نوفمبر) / ربيعي (فيفري-مارس) |
| **ولايات الإنتاج** | سعيدة، تيارت، تبسة، سوق أهراس، قسنطينة، ميلة |
| **المساحة** | ~2 مليون هكتار |
| **الإنتاج السنوي** | ~30 مليون قنطار |
| **الأمراض** | الصدأ الأصفر (rouille jaune), التفحم (charbon), البياض الدقيقي (oïdium) |
| **الري** | معظمه بعل (مطري)، يحتاج 300-400 مم/موسم |
| **الأصناف** | HD 1220, Vitron, El Waha, Anza, Sahel |

## الشعير (Orge)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | أكتوبر-نوفمبر |
| **ولايات الإنتاج** | سعيدة، تيارت، الجلفة، المدية |
| **المساحة** | ~1.5 مليون هكتار |
| **الإنتاج السنوي** | ~15 مليون قنطار |
| **الاستعمال** | علف الحيوانات + استهلاك بشري محدود |
| **مقاومة** | أكثر مقاومة للجفاف من القمح |

## الطماطم (Tomate)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | زراعة: فيفري-مارس، حصاد: جوان-أوت |
| **ولايات الإنتاج** | بليدة، البويرة، عين الدفلى، الشلف، مستغانم، سكيكدة |
| **المساحة** | ~30,000 هكتار |
| **الإنتاج السنوي** | ~12 مليون قنطار |
| **أنواع** | طماطم أكل (fraîche)، طماطم صناعية (industrielle، للمعلّب) |
| **الأمراض** | Mildiou (البياض الزغبي), Oïdium (البياض الدقيقي), Alternaria, Fusarium, Septoria |
| **الآفات** | Tuta absoluta (حفار أوراق الطماطم), Pucerons (المن), Aleurodes (الذبابة البيضاء) |
| **الري** | بالتنقيط (goutte-à-goutte) — ضروري |
| **التسميد** | NPK 20-20-20 أساسي + تسميد بالبوتاسيوم في مرحلة الإثمار |

## البطاطا (Pomme de terre)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | زراعة: جانفي-فيفري، حصاد: ماي-جوان |
| **ولايات الإنتاج** | عين مليلة، سطيف، البويرة، بومرداس، تسمسيلت |
| **المساحة** | ~60,000 هكتار |
| **الإنتاج السنوي** | ~30 مليون قنطار |
| **الأمراض** | Mildiou (البياض الزغبي), Alternaria, Rhizoctone, Gale commune |
| **الآفات** | Doryphore (خنفساء كولورادو), Taupin |
| **الأصناف** | Spunta, Désirée, Kondor, Nicola, Atlas |
| **التخزين** | في مخازن مبردة (3-5°C) للحفاظ على الجودة |

## الزيتون (Olivier)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | حصاد: أكتوبر-نوفمبر |
| **ولايات الإنتاج** | بجاية، تيزي وزو، البويرة، سطيف، جيجل، بومرداس |
| **عدد الأشجار** | ~35 مليون شجرة |
| **المساحة** | ~450,000 هكتار |
| **الإنتاج السنوي** | ~5 ملايين قنطار زيتون (60-70 ألف طن زيت) |
| **الأصناف** | Chemlal (الأكثر), Sigoise, Azeradj, Blanquette, Rougette, Limli |
| **الأمراض** | Verticilliose, Œil de paon (عيون الطاووس), Mouche de l'olive |
| **التربة** | تربة كلسية-طينية، تحتاج تصريف جيد |
| **التسميد** | نيتروجين في الربيع، بوتاسيوم في الخريف |
| **التقليم** | تقليم سنوي في الشتاء (ديسمبر-فيفري) |

## التمر (Datte)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | حصاد: سبتمبر-نوفمبر |
| **ولايات الإنتاج** | ورقلة، بسكرة، أدرار، غرداية، المغير، تقرت، إيليزي |
| **عدد النخيل** | ~20 مليون نخلة |
| **الإنتاج السنوي** | ~1 مليون قنطار |
| **أشهر الأصناف** | Deglet Nour (الأشهر), Ghars, Mehoul, Degla Baida, Timgouch |
| **الأمراض** | Boufaroua (البياض الدقيقي), Khamedj (التعفن), Fusariose |
| **الآفات** | Ver de la datte (دودة التمر), Cochenille (الحشرات القشرية), Araignée rouge |
| **التلقيح** | تلقيح يدوي في فيفري-مارس |
| **الري** | ري بالغمر (500-1000 لتر/نخلة/أسبوع) — أو بالتنقيط الحديث |

## الحمضيات (Agrumes)

| المعلومة | القيمة |
|---------|--------|
| **الموسم** | حصاد: أكتوبر-جانفي |
| **ولايات الإنتاج** | بليدة، الشلف، مستغانم، بومرداس، تيبازة |
| **المساحة** | ~45,000 هكتار |
| **الإنتاج السنوي** | ~9 ملايين قنطار |
| **الأنواع** | برتقال: Washington Navel, Thomson; ليمون: Eureka; كليمنتين: Cadoux |
| **الأمراض** | Gommose (التقرح الصمغي), Phytophthora, Tristeza, Mal secco |
| **الآفات** | Cochenille (الحشرات القشرية), Mineuse (حفار الأوراق), Pucerons |

## الفلفل (Poivron/Piment)

| المعلومة | القيمة |
|---------|--------|
| **ولايات الإنتاج** | بليدة، البويرة، جيجل، سكيكدة |
| **الموسم** | مارس-أبريل → أوت-أكتوبر |
| **الأمراض** | Phytophthora, Oïdium, Botrytis, Virus de la mosaïque |
| **الآفات** | Pucerons, Thrips, Aleurodes |

## البصل (Oignon)

| المعلومة | القيمة |
|---------|--------|
| **ولايات الإنتاج** | ميلة، سوق أهراس، قالمة، البويرة |
| **الموسم** | زراعة: سبتمبر-أكتوبر (شتوي) / فيفري-مارس (ربيعي) |
| **الإنتاج السنوي** | ~10 ملايين قنطار |
| **الأمراض** | Mildiou, Botrytis, Mouche de l'oignon |

## البطيخ (Pastèque/Melon)

| المعلومة | القيمة |
|---------|--------|
| **ولايات الإنتاج** | بسكرة، وادي سوف، غرداية، المدية |
| **الموسم** | مارس-أبريل → جوان-أوت |
| **الري** | بالتنقيط (ضروري جداً للبطيخ) |
| **الأمراض** | Fusarium, Oïdium, Puceron |
