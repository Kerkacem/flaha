## 🇹🇳 تونس — التوسع الأول (السنة 2-3)

### لماذا تونس أولاً؟
1. **نفس اللغة** (الدارجة التونسية قريبة جداً)
2. **نفس المناخ** (متوسطي مثل الجزائر)
3. **نظام فلاحي مشابه** (فلاحون صغار, زيتون, قمح)
4. **اشتغلت منصة مشابهة**: Mashroo3i (سوق B2B تونسي)
5. **الجالية التونسية** 1.5M في الخارج

### تكييف فلاحة لتونس
| العنصر | التغيير المطلوب |
|--------|----------------|
| اللهجة | دارجة تونسية (كلمات مختلفة) |
| الدفع | Poste Tunisienne + Flip |
| المحاصيل | زيتون (Chemlali), قمح, فلفل |
| البنوك | BNA, BTS |
| الشحن | Tunisie Poste + DHL |
| الحكومة | APIA (وكالة النهوض الفلاحي) |

### المحاصيل التونسية الأساسية
- **زيت الزيتون**: Chemlali, Chetoui — 200K طن/سنة (مصدر رئيسي)
- **التمور**: دقلة نور (جنوب تونس)
- **القمح**: بعل (شمال تونس)
- **الحمضيات**: برتقال (الوطن القبلي)

### إيرادات متوقعة في تونس
| المصدر | السنة 1 (تونس) | السنة 2 (تونس) |
|--------|---------------|---------------|
| السوق | $200K | $1.0M |
| الجالية التونسية | $150K | $800K |
| المدخلات | $100K | $500K |
| التمويل | $50K | $200K |
| **الإجمالي** | **$500K** | **$2.5M** |

## 🇲🇦 المغرب — التوسع الثاني (السنة 2-3)

### لماذا المغرب؟
1. **أكبر اقتصاد فلاحي** في شمال أفريقيا ($12B)
2. **تصدير كبير**: طماطم, فلفل, خضروات إلى أوروبا
3. **نمو رقمي سريع**: 90% إنترنت, 50%+ يستعملون الدفع الرقمي
4. **منافس محتمل**: ElKhadra — لكن فلاحة أوسع

### تكييف فلاحة للمغرب
| العنصر | التغيير المطلوب |
|--------|----------------|
| اللهجة | دارجة مغربية (مختلفة شويا) |
| الدفع | M-Wallet (Mobicash, Orange Money), CIH Bank |
| المحاصيل | طماطم (للتصدير), فلفل, زيتون, أرجان |
| البنوك | Crédit Agricole du Maroc, CIH, BC |
| الشحن | Poste Maroc + DHL + شركات محلية |
| الحكومة | ADA (وكالة التنمية الفلاحية), Green Morocco Plan |

### المحاصيل المغربية الأساسية
- **الطماطم**: 1.2M طن/سنة — الأولى في أفريقيا (تصدير لأوروبا)
- **زيت الأرجان**: $300-500/لتر — منتج فاخر
- **الحمضيات**: برتقال, كلومنتين
- **الفلفل**: 700K طن/سنة

### تحدي ElKhadra
- ElKhadra: منصة B2B مغربية للمنتجات الفلاحية
- حجمها صغير (مقارنة بـ DeHaat)
- **نقطة ضعفها**: لا تمويل, لا إرشاد, لا جالية
- **فلاحة تتفوق**: خدمات أشمل

### إيرادات متوقعة في المغرب
| المصدر | السنة 1 (المغرب) | السنة 2 (المغرب) |
|--------|-----------------|-----------------|
| السوق | $300K | $1.5M |
| الجالية المغربية | $200K | $1.0M |
| المدخلات | $150K | $600K |
| التمويل | $75K | $300K |
| **الإجمالي** | **$725K** | **$3.4M** |
