# MCP Tool: Payment Baridi

## الوصف
بوابة الدفع عبر بريدي موب (Baridi Mob) — نظام الدفع الأكثر انتشاراً في الجزائر.

## التقنية
- **المصدر**: AlBarid Bank API / Baridi Mob API
- **النوع**: REST API

## الأوامر

### initiate_payment
بدء عملية دفع.

```json
{
  "from": "+213XXXXXXXXX",  // رقم هاتف المشتري (بريدي موب)
  "amount": 5000,
  "reference": "FL-2026-001",
  "description": "زيت زيتون بكري 5 لترات"
}
```

### verify_payment
التحقق من الدفع.

```json
{
  "transaction_id": "BAR-XXXXX",
  "reference": "FL-2026-001"
}
```

### transfer_to_farmer
تحويل المبلغ للفلاح.

```json
{
  "farmer_baridi": "XXXXXXXXX",  // رقم بريدي موب للفلاح
  "amount": 4750,  // بعد خصم العمولة
  "reference": "FL-2026-001"
}
```

### check_balance
استعلام عن الرصيد.

## طرق الدفع المدعومة
1. **بريدي موب (Baridi Mob)** — الأكثر انتشاراً
2. **CCP (Chèques Postaux)** — للحوالات الكبيرة
3. **Edahabia** — بطاقة ذهبية (للمشتريين من الجالية)
4. **Western Union** — للجالية بالخارج
5. **Wise/Revolut** — تحويل دولي للجالية

## المميزات
- دعم الفلاحين بدون حسابات بنكية (يكفي بريدي موب)
- دفع آمن: المبلغ في حساب ضمان حتى تأكيد الاستلام
- تحويل تلقائي للفلاح بعد تأكيد التسليم
- عمولة منخفضة (1-2% مقارنة بـ 5% في الوسائل التقليدية)
