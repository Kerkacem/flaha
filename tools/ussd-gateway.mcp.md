# MCP Tool: USSD Gateway

## الوصف
بوابة USSD للفلاحين الذين لا يملكون هواتف ذكية أو أنترنت. يكفي أي هاتف عادي. مستوحى من Twiga Foods (كينيا).

## التقنية
- **المصدر**: USSD Gateway (اتصالات الجزائر، موبيليس، أوريدو)
- **النوع**: USSD Protocol

## الرمز
- `*فلاحة#` ← القائمة الرئيسية
- `*فلاحة*1#` ← بيع منتج
- `*فلاحة*2#` ← شراء منتج
- `*فلاحة*3#` ← أسعار السوق
- `*فلاحة*4#` ← استشارة فلاحية
- `*فلاحة*5#` ← الدعم الحكومي

## القائمة الرئيسية (USSD)

```
🌾 فْلاحة
1️⃣ بيع منتج
2️⃣ شراء منتج
3️⃣ أسعار السوق
4️⃣ استشارة
5️⃣ الدعم الحكومي
6️⃣ رصيدي
#
```

## سير عمل البيع عبر USSD

```
الفلاح يطلب: *فلاحة*1#
→ الرد: أدخل المنتج (1-طماطم, 2-بطاطا, 3-فلفل, 4-زيتون...)
→ الفلاح: 1
→ الرد: أدخل الكمية (قناطير)
→ الفلاح: 5
→ الرد: أدخل السعر (دج)
→ الفلاح: 2000
→ الرد: تم تسجيل إعلانك. رمز المتابعة: FL-2026-001
```
