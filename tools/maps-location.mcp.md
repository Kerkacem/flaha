# MCP Tool: Maps & Location

## الوصف
خرائط ومواقع لتحديد نقاط التجميع، أقرب الموردين، وحساب مسافات التوصيل.

## التقنية
- **المصدر**: Google Maps API / OpenStreetMap (مجاني)
- **النوع**: REST API

## الأوامر

### find_nearest_collection_point
أقرب نقطة تجميع للفلاح.

```json
{
  "farmer_location": "سوق الخميس, البويرة",
  "radius_km": 20
}
```

### calculate_distance
حساب مسافة التوصيل.

```json
{
  "from": "البويرة",
  "to": "الجزائر العاصمة"
}
```

### find_suppliers
أقرب موردي المدخلات الفلاحية.

```json
{
  "product": "مبيد X",
  "near": "البويرة",
  "radius_km": 50
}
```

### optimize_route
أفضل طريق لجمع المنتجات من عدة فلاحين.

```json
{
  "pickups": ["البلدية 1", "البلدية 2", "البلدية 3"],
  "destination": "سوق الجملة الجزائر"
}
```

## نقاط التجميع المقترحة
- نقاط التجميع في كل بلدية (مساجد، أسواق، محلات متعاونين)
- تظهر على الخريطة للفلاحين
- نظام حجز: الفلاح يحجز وقت التسليم في نقطة التجميع
