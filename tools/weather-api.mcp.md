# MCP Tool: Weather API

## الوصف
الحصول على بيانات الطقس والمناخ في الجزائر. متكامل مع ONM (ديوان الأرصاد الجوية).

## التقنية
- **المصدر**: OpenWeatherMap API + ONM (Office National de la Météorologie)
- **النوع**: REST API

## الأوامر

### get_weather
طقس اليوم في منطقة معينة.

```json
{
  "wilaya": "البويرة",
  "days": 7
}
```

### get_forecast
توقعات الطقس للأيام القادمة.

```json
{
  "wilaya": "بليدة",
  "days": 15
}
```

### get_agricultural_advice
توصيات فلاحية حسب الطقس.

```json
{
  "wilaya": "سطيف",
  "crop": "potato",
  "growth_stage": "flowering"
}
```

النتيجة:
```json
{
  "advice": "متوقع ارتفاع في الحرارة يوم الأربعاء (38°C) — أنصح بتأخير السقي لوقت المغرب",
  "alerts": ["رياح قوية يوم الخميس — احمي الأشجار"]
}
```

## التغطية
- جميع 58 ولاية جزائرية
- توقعات 15 يوم
- تنبيهات الطقس القاسي (برد، رياح، أمطار غزيرة)
