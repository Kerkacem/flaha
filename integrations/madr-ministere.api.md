# تكامل وزارة الفلاحة والتنمية الريفية

## الوصف
ربط منصة فلاحة مع وزارة الفلاحة الجزائرية لأتمتة إجراءات الدعم الحكومي والتسجيل.

## الخدمات

### 1. التحقق من بطاقة الفلاح
- **الوصف**: التحقق من صحة بطاقة الفلاح عبر قاعدة البيانات الوطنية
- **المدخلات**: رقم البطاقة، اسم الفلاح، تاريخ الميلاد
- **المخرجات**: صالحة/غير صالحة + تاريخ الانتهاء

### 2. تقديم ملفات الدعم إلكترونياً
- **الوصف**: رفع وإرسال ملفات الدعم (FNDIA, FGVA) إلكترونياً
- **المدخلات**: ملف الفلاح الرقمي + الوثائق المطلوبة
- **المخرجات**: رقم الطلب + تاريخ المعالجة

### 3. متابعة حالة الطلب
- **الوصف**: استعلام عن حالة طلب الدعم
- **المدخلات**: رقم الطلب
- **المخرجات**: حالة الطلب + تعليقات

### 4. بيانات إحصائية
- **الوصف**: الحصول على إحصائيات الإنتاج الفلاحي
- **المدخلات**: الولاية، الموسم، المحصول
- **المخرجات**: المساحة، الإنتاج، الإنتاجية
