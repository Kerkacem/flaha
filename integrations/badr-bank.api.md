# تكامل بنك BADR (بنك التنمية الريفية)

## الوصف
ربط منصة فلاحة مع بنك BADR لتمويل الفلاحين وقروض الموسم الفلاحي.

## الخدمات المقدمة عبر المنصة

### 1. قرض RFIG (بدون فوائد)
- **المبلغ**: حتى 1,000,000 دج
- **المدة**: 5 سنوات
- **الفائدة**: 0% (بدون فوائد)
- **الشروط**:
  - الفلاح مسجل في المنصة (سجل رقمي)
  - أقل من 10 هكتار
  - سجل مبيعات إيجابي (6 أشهر على الأقل)

### 2. قرض الموسم الفلاحي
- **الغرض**: شراء بذور، أسمدة، مبيدات
- **المبلغ**: حتى 500,000 دج
- **السداد**: بعد الحصاد (6-8 أشهر)
- **الضمان**: المحصول المستقبلي

### 3. تمويل المعدات الفلاحية
- **الغرض**: شراء مضخات، دفيئات، معدات ري
- **المبلغ**: حتى 2,000,000 دج
- **المدة**: حتى 7 سنوات

## سير العمل
1. الفلاح يطلب قرض عبر المنصة
2. المنصة تولد "سجل الفلاح الرقمي"
3. المنصة ترسل الطلب لـ BADR
4. BADR يحلل السجل ويوافق (خلال 48 ساعة)
5. المبلغ يحول لحساب الفلاح
6. المنصة تتابع السداد

## API
- **المصادقة**: API Key + شهادة رقمية
- **النوع**: REST API
- **البيئة**: test.badr.dz / api.badr.dz
