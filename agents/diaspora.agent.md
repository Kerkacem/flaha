# 🌍 وكيل الجالية — Diaspora Agent

## الوصف
وكيل الجالية يربط 6 ملايين جزائري في الخارج بالفلاحين في الجزائر. فكرة جزائرية خالصة (لا توجد في أي منصة عالمية).

## منتجات مطلوبة من الجالية

| المنتج | الطلب | الدول المستهدفة | السعر التقريبي (تصدير) |
|--------|-------|-----------------|------------------------|
| **زيت زيتون بكري** | عالي جداً | فرنسا، كندا، إسبانيا | 5000-8000 دج/لتر |
| **تمر دقلة نور** | عالي | فرنسا، كندا، أمريكا | 3000-5000 دج/كغ |
| **عسل بلدي** | عالي | فرنسا، كندا | 4000-7000 دج/كغ |
| **كسكس بلدي** | متوسط | فرنسا، بلجيكا | 1500-2000 دج/كغ |
| **الملفوف (مشروبات)** | متوسط | فرنسا | 2000-3000 دج/لتر |
| **زعتر بلدي** | متوسط | كل الدول | 1000-1500 دج/كغ |
| **توابل بلدي** | عالي | فرنسا، كندا | 800-1200 دج/كغ |
| **مربى بلدي** | متوسط | فرنسا | 500-1000 دج/مرطبان |

## سير العمل

### للجالية (الطلب):
1. الزبون يرسل واتساب: "نبي زيت زيتون بكري 10 لترات"
2. الوكيل يعرض الخيارات: فلاحين مختلفين، أسعار، جودة (بالصور)
3. الزبون يختار ويدفع (Baridi Mob / Western Union / TransferWise)
4. الوكيل ينسق مع الفلاح للتوصيل
5. الوكيل يتابع الشحنة: التخليص الجمركي → الشحن → التوصيل
6. تأكيد الاستلام وتقييم

### للفلاح (التصدير):
1. الفلاح يسجل منتوجه للتصدير
2. الوكيل ينشئ ملف تصدير: الكمية، السعر، الشهادات
3. الوكيل يبحث عن مشترين في الخارج
4. ينسق الشحن الدولي (بريد الجزائر، DHL، FedEx)
5. يتابع الدفع بالعملة الصعبة

## التحديات والحلول
- **الجمركة**: الوكيل يقدم الوثائق المطلوبة آلياً
- **الدفع بالعملة الصعبة**: عبر Wise, Revolut, Western Union
- **الثقة**: صور وفيديوهات من الفلاح + تقييمات
- **الجودة**: الفلاح يضمن الجودة والوكيل يتابع

## التكامل
- التسويق: إعلانات مستهدفة للجالية
- اللوجستيك: شركات شحن دولي
- السوق: تصدير منتجات الفلاحين الجزائريين
