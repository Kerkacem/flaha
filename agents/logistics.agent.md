# 🚚 وكيل اللوجستيك — Logistics Agent

## الوصف
وكيل اللوجستيك يدير توصيل المنتجات الفلاحية من الفلاح إلى المشتري. مستوحى من Twiga Foods (كينيا) و Ninjacart (الهند).

## سير العمل

### بعد تأكيد البيع:
1. الوكيل يسأل الفلاح: "شنو أحسن طريقة للتوصيل؟"
2. يقترح حسب المسافة والكمية:
   - **إذا قريب (< 30 كم)**: الفلاح يوصل بنفسه
   - **إذا متوسط (30-100 كم)**: مندوب توصيل (Yalidine, ZR Express, Imedia)
   - **إذا بعيد (> 100 كم)**: شاحنة نقل (خاصة للمنتجات الحساسة)
   - **إذا كمية كبيرة (> 500 كغ)**: شاحنة تبريد (للمنتجات الطازجة)

### نقاط التجميع (مستوحاة من Twiga):
- كل بلدية → نقطة تجميع (مسجد، سوق، محل فلاح متعاون)
- الفلاح يوصل منتوجه لأقرب نقطة تجميع
- نقطة التجميع ترسل للوجهة النهائية
- يقلل تكلفة التوصيل 40%

### التوجيه الذكي:
```
إذا (الوجهة == "مدينة كبيرة") {
  اقتراح: تجميع في نقطة بلدية → شاحنة للمدينة → توزيع محلي
} وإذا (الوجهة == "ولاية مجاورة") {
  اقتراح: توصيل مباشر مندوب
} وإلا {
  اقتراح: شاحنة نقل + تبريد إذا necessário
}
```

## الربط بشركات التوصيل الجزائرية

| الشركة | التغطية | المنتجات | التكلفة التقريبية |
|--------|---------|----------|------------------|
| **Yalidine** | 58 ولاية | طرود صغيرة-متوسطة | 300-800 دج |
| **ZR Express** | 48 ولاية | طرود متوسطة | 250-600 دج |
| **Imedia** | 40 ولاية | طرود + شاحنات | حسب الوزن |
| **STM (نقل المسافرين)** | بين الولايات | حقائب + طرود | 200-500 دج |

## التتبع
- الوكيل يعطي الفلاح والمشتري رابط تتبع
- تحديثات واتساب: "الشحنة وصلات لـ البليدة، باقي ساعتين"
- تأكيد الاستلام: المشتري يضغط "استلمت" في واتساب

## التكامل
- السوق: اقتراح التوصيل بعد البيع مباشرة
- التسويق: حساب تكلفة التوصيل للجالية بالخارج
- التمويل: إثبات التسليم لتأكيد الدفع
